# chatgpt-cli

通过命令行与 ChatGPT 对话

## 使用

构建

```shell
go build github.com/keybrl/chatgpt-cli
```

运行

```shell
chatgpt-cli chat --secret-key '<Your-OpenAI-API-Key>'
```

或者直接构建并运行

```shell
go run github.com/keybrl/chatgpt-cli chat --secret-key '<Your-OpenAI-API-Key>'
```

更多使用方法参考：

```shell
chatgpt-cli --help
```
